# ARGON124 - Сайт-лендинг для услуг аргонной сварки

![Аргонная сварка в Красноярске](https://argon124.ru/img/og-preview.jpg)

## 📋 О проекте

ARGON124 — это одностраничный сайт-визитка (лендинг) для мастерской аргонной сварки в Красноярске. Сайт разработан для привлечения клиентов и демонстрации услуг компании, которая работает на рынке с 2005 года.

**Ссылка на сайт:** [https://argon124.ru](https://argon124.ru)

---

## 🎯 Целевая аудитория

- Владельцы автомобилей (ремонт радиаторов, проточка дисков)
- Частные лица и организации, нуждающиеся в сварке цветных металлов
- Автосервисы и гаражные мастерские

## ✨ Функциональность

### Основные разделы:
- **Главная (Hero-секция)** — краткое представление и кнопки звонка
- **Преимущества** — 4 ключевых причины выбрать компанию
- **Услуги** — 10 видов услуг с модальными окнами (цены и описание)
- **Портфолио** — галерея работ (16 фото) с лайтбоксом для просмотра
- **Отзывы** — слайдер с отзывами клиентов
- **О нас** — информация о компании и статистика
- **Контакты** — адрес, телефоны, график работы, Яндекс.Карта

### Интерактивные элементы:
- ✅ Модальные окна с ценами для каждой услуги
- ✅ Лайтбокс для просмотра фото в портфолио
- ✅ Адаптивное мобильное меню
- ✅ Плавная прокрутка по якорным ссылкам
- ✅ Слайдеры (отзывы и портфолио)
- ✅ Подсветка активного раздела в меню

## 🚀 Установка и запуск локально

### Вариант 1: Простой (открыть в браузере)
1. Скачайте репозиторий:
   ```bash
   git clone https://github.com/yourusername/argon124.git
   ```
2. Откройте файл `index.html` в любом современном браузере

### Вариант 2: Локальный сервер (рекомендуется)
1. Установите расширение Live Server для VS Code
2. Откройте проект в VS Code
3. Нажмите "Go Live" в правом нижнем углу
4. Сайт откроется на `http://127.0.0.1:5500`

### Требования:
- Любой современный браузер (Chrome, Firefox, Safari, Edge)
- Для разработки: VS Code (рекомендуется)

## 💡 Особенности реализации

### Почему нет базы данных и форм обратной связи?
По требованию заказчика. Целевая аудитория (водители, автослесари) предпочитает **звонить по телефону**, а не ждать ответа на заявку. Все данные статичны и хранятся в HTML и JavaScript, что обеспечивает:
- Быструю загрузку
- Простоту хостинга
- Безопасность (нет хранения персональных данных)

### Хранение данных услуг:
Вся информация о ценах и составе услуг хранится в JavaScript-объекте `priceData` и подставляется в модальные окна динамически.
